# 멈춤 5초 — 챈트·노래·손동작 가이드

> 머리가 아닌 몸이 외우게 만드는 핵심 자산.
> 영상·강의·포스터 어디서나 동일하게 사용.

---

## 1. 마스터 챈트 (5박자)

> **"멈춰, 의심, 확인, 끊고, 알려."**

- 5음절 5박
- 한 박자에 한 손가락 접기
- 처음엔 천천히, 다섯 번째는 빠르게

### 박자 표

```
박자:   1     2      3      4      5
입:    멈춰  의심   확인   끊고   알려
손:    엄지  검지   중지   약지   새끼
       ↓     ↓      ↓      ↓      ↓
       (접) (접)   (접)   (접)   (접)
```

5박 끝나면 주먹이 만들어집니다. 주먹이 곧 "멈춤 완성".

---

## 2. 노래 (트로트풍 후렴)

### 가사 — "멈춤 5초"

```
1절
전화벨이 울려도   잠깐만요
가슴이 두근거려도  잠깐만요
손가락 다섯 개   하나씩 접으며

(후렴)
멈춰 ~ 의심 ~ 확인 ~
끊고 ~ 알려 ~
오초만 멈추면 평생을 지켜요
오초만 멈추면 우리 집을 지켜요

2절
"가족이 사고났대요"  잠깐만요
"지금 당장이래요"   잠깐만요
입으로 말해봐요    "이거 이상한데?"

(후렴 반복)

(엔딩)
잘못한 거 아니에요
사기범이 정교한 거예요
멈출 줄 아는 어르신이
가장 강한 어르신
```

### 멜로디 가이드

- 트로트 4/4박, 80~100 BPM
- 후렴은 5박 챈트와 동일한 리듬으로 (멈-춰-의-심-확-인…)
- 마지막 후렴은 다 같이 큰 소리로

### 활용

- 강의 시작·끝
- 카카오톡 음성메시지로 가족이 부모님께 보내기
- 복지관 운동시간 배경음악

## 3. 손동작 5단계 (그림 설명)

### 1단계 · 멈춤 — 엄지 접기

```
   ✋          🤚
  펼친 손       엄지만 접기
  (시작)       "잠깐만요"
```

- 손바닥을 펴서 상대에게 보여줌 (정지신호)
- 엄지를 손바닥 안으로 접으며 "**멈춰**"

### 2단계 · 의심 — 검지 접기

- 펴 있는 검지를 접으며 "**의심**"
- 동시에 입으로 소리내어 "이거 이상한데?"

### 3단계 · 확인 — 중지 접기

- 중지를 접으며 "**확인**"
- 다른 손으로 전화 거는 시늉을 함께

### 4단계 · 끊기 — 약지 접기

- 약지를 접으며 "**끊고**"
- 단호하게, 변명 없이

### 5단계 · 알리기 — 새끼 접기 → 주먹 완성

- 새끼를 접으며 "**알려**"
- 주먹이 완성됨 = 멈춤 완성
- 완성된 주먹을 가슴에 한 번 가져다 댐 ("나를 지켰다")

## 4. 응용 동작

### 짝꿍 동작 (2인용)

마주 보고 손바닥을 펴서 서로에게 보여줍니다.
"멈춰" 하며 동시에 엄지를 접고… 5단계 끝나면 주먹 vs 주먹으로 가볍게 인사.

### 합창 동작 (단체용, 강의장)

강사가 "준비!" 하면 모두 손바닥을 펴고,
"하나, 둘—" 하면 다 같이 챈트와 동작.

### 손주 동작 (가족용)

손주가 할머니 손을 잡고 같이 손가락을 접으며 가르쳐드림.
TikTok·릴스 챌린지화 가능: **#멈춤5초챌린지**

## 5. 영상 콘티 (15초 숏폼)

```
[0~2초]  화면 가득 펴진 손바닥 — 자막: "전화 받기 전에"
[2~4초]  손가락이 하나씩 접히기 시작 — 자막: "멈춰"
[4~6초]  두 번째 손가락 접힘 — 자막: "의심"
[6~8초]  세 번째 — 자막: "확인"
[8~10초] 네 번째 — 자막: "끊고"
[10~12초] 다섯 번째, 주먹 완성 — 자막: "알려"
[12~15초] 주먹을 가슴에 — 자막: "5초가 평생을 지킵니다"
          로고: 멈춤연습 stopcall.kr
```

배경음: 챈트 5박 + 트로트 후렴 일부

## 6. 인쇄 자산용 손가락 일러스트 명세

디자이너 의뢰서:

- 따뜻한 살구색 손, 노년층 특징(주름·반점) 살짝 살리되 미화하지 않기
- 5컷 시퀀스: 손바닥 → 엄지접힘 → … → 주먹
- 각 컷마다 손목 옆에 색띠 (빨강→노랑→파랑→초록→보라)
- 라인 두께 굵게 (시인성, 70대 독자 기준)
- 배경 투명 PNG / SVG
