# 11장. 주소를 쳐도 가짜로 가는 함정 — 파밍

## 📖 오늘의 한 문장
> **"주소를 정확히 쳤는데도 가짜일 수 있습니다. 자물쇠 모양부터 확인하세요."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **파밍(Pharming)** | 악성코드에 감염된 컴퓨터가 진짜 주소를 쳐도 가짜 사이트로 보내버리는 사기 |
| **악성코드** | 내 기기에 몰래 설치되어 나쁜 일을 하는 프로그램 |
| **자물쇠 모양(🔒)** | 주소창 왼쪽에 있는 표시. 안전한 연결이라는 뜻 |
| **공식 앱** | 은행·카드사에서 만든 진짜 앱. 파밍에 강해요 |

---

## 🌐 실제 사례 — 이만수 님(65세) 이야기

이 씨는 컴퓨터로 국민은행 사이트에 들어갔습니다. 주소창에 `kbstar.com`이라고 정확히 쳤습니다. 평소처럼 로그인 화면이 나왔고, 보안카드 번호 35개를 전부 입력하라는 창이 떴습니다. **"원래는 2개만 물어봤는데?"** 이상하다 싶어 멈췄습니다. 알고 보니 일주일 전 받은 이메일의 파일을 열어본 뒤로, 컴퓨터가 감염되어 **진짜 주소를 쳐도 가짜 은행 사이트로 연결**되고 있었습니다.

---

## ⚠️ 위험신호 — 이 사이트는 100% 가짜입니다

```
□ 평소와 다르게 "보안카드 번호 전부" 요구
□ 주소창 왼쪽 자물쇠(🔒)가 없거나 빨간 경고
□ 로그인 후 공인인증서·OTP를 "재발급" 하라고 함
□ 갑자기 화면 디자인이 살짝 어색함
□ "보안 프로그램 설치" 팝업이 계속 뜸
```

> **은행은 보안카드 번호 전부를 한 번에 묻지 않습니다.** 이상하면 바로 나와서 전화로 확인하세요.

---

## 🛡️ "입력 전 1분" 체크 3단계

1. **주소창 자물쇠(🔒)가 있는가?** — 없거나 빨간색이면 바로 닫기
2. **요구하는 정보가 평소와 같은가?** — 보안카드 번호 전부는 절대 X
3. **공식 앱으로 다시 들어가 보기** — 은행 앱에서 같은 화면이 보이는지

---

## ✏️ 활동 ① — 진짜/가짜 사이트 가려내기

| 상황 | 진짜 / 가짜 |
|---|---|
| 주소창에 `kbstar.com`, 🔒 표시, 평소처럼 보안카드 2개만 요구 | ☐ |
| 주소창에 `kbstar.com`, 🔒 없음, 보안카드 35개 전부 요구 | ☐ |
| 주소창에 `kb-star.com`, 🔒 있지만 로고가 살짝 다름 | ☐ |
| 국민은행 **공식 앱(KB스타뱅킹)**에서 지문으로 로그인 | ☐ |
| 팝업창이 계속 뜨면서 "보안 프로그램 설치" 요구 | ☐ |

> *정답: 진짜 / 가짜 / 가짜(주소 철자 다름) / 진짜 / 가짜*
> 핵심: **자물쇠(🔒) + 평소와 같은 흐름** 둘 다 맞아야 안전.

---

## ✏️ 활동 ② — 파밍 예방 습관 3가지

```
[1] 은행 업무는 되도록 공식 앱으로 → PC보다 안전
[2] 출처 불명 파일·첨부문서 열지 않기
[3] 백신 프로그램 최신 상태로 유지 (자동 업데이트 켜기)
```

---

## 🆘 만약 보안카드 정보를 입력했다면

```
1. 인터넷 선 뽑기 / 와이파이 끄기
2. 은행 고객센터 전화 (국민은행 1588-9999)
   → "파밍 피해" 라고 말하고 계좌 지급정지 요청
3. 경찰청 사이버수사국 ☎ 182
4. 보안카드 / OTP / 공인인증서 전부 재발급
5. 컴퓨터는 전문가에게 맡겨 포맷 또는 복원
```

> 📌 **지급정지가 30분 안에 이뤄지면 대부분 돈을 지킵니다.**
