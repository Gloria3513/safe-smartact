# 멈춤 5초 — 영상 스크립트 (30초 / 3분)

> 공식 영상 콘텐츠 마스터 스크립트.
> 모든 영상의 마지막은 챈트 + 로고로 통일.

---

## A. 30초 숏폼 (유튜브 쇼츠 / 인스타 릴스 / 틱톡)

### 컨셉
시니어 모델 1명이 정면을 보며 손가락 5개로 5단계를 천천히 시연.
자막은 큰 글씨 + 색띠.

### 스크립트

```
[0~3초] 시니어 모델 정면, 펴진 손
모델: "전화 받기 전에, 5초만 멈춰요."
자막: 5초가 평생을 지킵니다

[3~6초] 엄지 접으며
모델: "멈춰."
자막: 멈춰 (빨강)

[6~9초] 검지 접으며
모델: "의심."
자막: 의심 (노랑)

[9~12초] 중지 접으며
모델: "확인."
자막: 확인 (파랑)

[12~15초] 약지 접으며
모델: "끊고."
자막: 끊고 (초록)

[15~18초] 새끼 접으며 (주먹 완성)
모델: "알려."
자막: 알려 (보라)

[18~22초] 주먹을 가슴에
모델: "잘못한 거 아니에요. 사기범이 너무 정교한 거예요."
자막: (멘트 그대로)

[22~27초] 다 같이 챈트
화면: 다양한 시니어 5명 컷편집, 다 같이 챈트
자막: "멈춰, 의심, 확인, 끊고, 알려"

[27~30초] 로고
자막: 멈춤연습 · stopcall.kr
       오늘 1분만 연습해보세요
```

### 촬영 노트
- 모델: 70대 시니어, 본인 손이 잘 보이게
- 배경: 단색 (베이지·연한 노랑)
- 조명: 부드럽게, 얼굴·손 모두 잘 보이게
- 음성: 또렷하고 천천히. BGM은 챈트 부분만 작게.

## B. 3분 강의 영상 (사이트 임베드 / 복지관 상영용)

### 제목
"5초가 평생을 지킵니다 — 멈춤 5초 루틴"

### 구성

```
[0:00~0:20] 오프닝
- 강사: "안녕하세요. 오늘 외울 거 없어요. 손가락 다섯 개만 있으면 됩니다."
- 자막: 멈춤연습 1강 / 5초가 평생을 지킵니다

[0:20~0:50] 순자 어머니 이야기 (30초)
- 내레이션 + 일러스트 또는 재연
- "어느 봄날 점심 직후, 순자 어머니는 전화를 받았습니다…"
- 결말: "5초가 200만 원을 지켰습니다."

[0:50~1:30] 5단계 시연 (40초)
- 강사 손 클로즈업
- "멈춰" — 엄지 / 자막
- "의심" — 검지 / 자막
- "확인" — 중지 / 자막
- "끊고" — 약지 / 자막
- "알려" — 새끼 / 자막 (주먹 완성)

[1:30~2:00] 3가지 미끼 신호 (30초)
- 강사: "이 세 단어 중 하나만 나와도 무조건 멈춤이에요."
- 카드 1: "가족이 사고래요" → 감정
- 카드 2: "처벌받는대요" → 공포
- 카드 3: "지금 당장이래요" → 긴급

[2:00~2:30] 본인확인 암호 (30초)
- 강사: "두 번째로 가져가실 보물이에요. 우리 집 암호."
- 예시 화면: "우리 강아지 이름 뭐였지?"
- "오늘 가족과 한 가지 정해두세요."

[2:30~2:50] 다 함께 챈트 (20초)
- 강사 + 시니어 모델 5명 합창
- "멈춰, 의심, 확인, 끊고, 알려" × 3회

[2:50~3:00] 클로징
- 강사: "잘못한 거 아니에요. 사기범이 너무 정교한 거예요."
- 자막: 5초가 평생을 지킵니다
- 로고: 멈춤연습 · stopcall.kr
```

### 자막 규칙
- 폰트: 굵은 고딕 (Pretendard Bold 이상)
- 크기: 화면 높이의 8% 이상
- 색: 흰색 + 검정 외곽선 (가독성)
- 5단계 자막은 항상 5색 컬러띠 동반

### 음향
- BGM: 잔잔, -20dB 이하
- 챈트 구간: BGM 멈추고 목소리만
- 마지막 클로징: 챈트 후 1초 무음 → 로고 사운드

## C. 60초 가족 캠페인 영상 (자녀가 부모님께 보내는)

### 컨셉
30대 자녀가 70대 부모에게 영상편지로 가르쳐드림.

### 스크립트

```
[0~10초] 자녀 정면, 카메라 응시
"엄마, 아빠. 오늘 5초만 시간 내주세요.
요즘 너무 정교한 사기 전화가 많아서요.
부탁이에요."

[10~30초] 손가락 시연 (자녀가 시범)
"손가락 다섯 개 펴주세요. 저랑 같이 해요.
멈춰… 의심… 확인… 끊고… 알려…
한 번 더요. 멈춰, 의심, 확인, 끊고, 알려."

[30~45초] 본인확인 암호 제안
"우리 둘만 아는 암호 하나 정해요.
이상한 전화 오면 그거 물어봐요.
나는 ___ 라고 답할게요."

[45~60초] 마무리
"잘못한 거 아니에요. 사기범이 너무 정교한 거예요.
엄마, 아빠 아끼는 사람이 곁에 있어요.
의심나면 무조건 저한테 먼저 전화해요. 사랑해요."

자막: 멈춤연습 · stopcall.kr
      가족이 함께하는 사이버안전
```

### 활용 방식
- 자녀가 직접 찍어서 부모님 카톡으로 보내기 (캠페인 템플릿 제공)
- 사이트에서 "내 영상 만들기" 유도
