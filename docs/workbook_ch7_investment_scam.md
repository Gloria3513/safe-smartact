# 7장. "이거만 사면 인생이 바뀝니다" — 리딩방·코인 투자 사기

## 📖 오늘의 한 문장
> **"확실한 투자는 세상에 없습니다. 확실하다고 하는 사람만 있을 뿐입니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **리딩방** | 단톡방·텔레그램에서 "오늘 이 종목 사세요" 알려주는 방. 99%가 사기 |
| **선생님/리더** | 리딩방의 우두머리. 전문가처럼 보이지만 대부분 가짜 |
| **코인 사기** | 가짜 거래소·가짜 코인으로 돈을 빼앗는 수법 |
| **출금 정지** | 처음엔 수익이 나는 것처럼 보이다가, 큰 돈을 넣으면 출금이 막히는 함정 |
| **카톡 단톡 미끼** | 모르는 단톡방에 초대돼 "여기 들어오면 돈 번대요" 권유받는 것 |

---

## 🎙️ 실제 사례 — 이정복 님(70세) 이야기

은퇴 후 적적하던 어느 날, 카카오톡으로 모르는 사람의 친구추가 알림이 왔습니다. **친절한 30대 여성** 프로필이었습니다.

> *"안녕하세요 어르신, 저 ○○증권 김민지 애널리스트예요. 좋은 정보 공유드리려고요."*

며칠 대화하다 보니 **"VIP 단톡방"**에 초대됐습니다. 거기엔 50명이 있었고, 매일 "오늘 ○○주 사세요" 하면 정말로 다음 날 오르고 있었습니다.

> *"여러분 오늘도 +12% 수익! 이번엔 더 큰 종목 갑니다. 1,000만 원 이상 투자 가능하신 분만 따로 안내."*

이 씨는 1,500만 원을 입금했습니다. 처음엔 잔고가 2,300만 원으로 늘어나는 게 보였습니다. 출금하려 했더니—

> *"수익금 출금은 세금 600만 원을 먼저 내셔야 가능합니다."*

세금 600만 원을 더 송금하자, 이번엔—

> *"본인확인 보증금 800만 원이 더 필요합니다."*

그제서야 이 씨는 깨달았습니다. **단톡방 50명 중 49명이 사기범**이었고, 잔고 화면도 가짜였습니다. **총 피해액 2,900만 원**.

---

## ⚠️ 위험신호 — 투자사기는 이렇게 걸려옵니다

```
□ 모르는 사람이 카톡·문자·DM으로 친구추가
□ "○○증권 애널리스트", "○○자산운용 부장" 자기소개
□ "좋은 정보가 있다" / "VIP 단톡방 초대"
□ 단톡방에 50~200명, 모두 "수익 났어요!" 인증샷
□ "확실하다", "절대 손해 안 본다", "정부 인증 사업"
□ "오늘만 가능", "이번 기회 놓치면 후회"
□ 처음엔 작은 금액부터 → 점점 큰 금액 유도
□ 수익은 보이는데 출금이 안 됨
□ 출금하려면 "세금", "보증금", "수수료" 더 내라고 함
```

> **이 중 2개 이상 해당되면 100% 사기입니다.** 진짜 금융기관은 카톡으로 종목 찍어주지 않습니다.

---

## 🧠 왜 똑똑한 사람도 속을까?

### ① 처음에는 진짜 수익을 보여줍니다

사기범은 **처음 한두 번은 진짜로 오를 종목**을 찍어줍니다. 또는 가짜 화면에서 "내 잔고가 늘어나는 모습"을 보여줍니다. 인간의 뇌는 한 번 성공을 맛보면 의심을 멈춥니다.

### ② 단톡방의 "다른 사람들"이 다 사기범입니다

50명짜리 단톡방에서 49명이 "감사합니다!" "오늘도 수익 났어요!" 라고 하면, **나만 의심하기 어렵습니다.** 그게 함정입니다. 그 49명이 다 사기범입니다.

### ③ "은퇴 후의 적적함"을 노립니다

처음엔 투자 얘기가 아닙니다. 안부·건강·취미 이야기로 친해진 다음 슬쩍 투자 이야기를 꺼냅니다. **외로움이 첫 번째 미끼**입니다.

### ④ "이번 한 번만 더"의 함정

1,500만 원을 잃었으니 600만 원 더 넣어서 회수하고 싶은 마음 — 사기범은 정확히 이 심리를 노립니다. 한 번 발을 들이면 끝까지 빠져나오기 어렵습니다.

## 🛡️ 절대 무너지지 않는 4가지 원칙

### 원칙 1 — 모르는 사람이 카톡으로 권하는 투자는 100% 사기

```
✓ 진짜 증권사 직원은 카톡으로 종목 안 찍어줍니다
✓ 진짜 애널리스트는 단톡방에서 활동하지 않습니다
✓ 진짜 금융기관은 친구추가로 영업하지 않습니다
```

### 원칙 2 — "확실하다"는 말이 나오면 무조건 사기

```
✓ "원금 보장" — 사기
✓ "절대 손해 안 본다" — 사기
✓ "정부가 인증한다" — 사기
✓ "내부 정보다" — 사기 (또는 불법)
```

### 원칙 3 — 출금이 한 번에 안 되면 그 돈은 끝났다고 생각하기

```
✓ "세금 먼저 내세요" → 사기
✓ "보증금 입금하세요" → 사기
✓ "본인확인 수수료" → 사기
✓ 진짜 거래소는 수익에서 세금을 알아서 떼고 줍니다
```

### 원칙 4 — 돈 얘기는 가족과 의논 후 결정

```
✓ "이번 기회 놓치면 안 된다"는 말이 가장 위험
✓ 진짜 좋은 기회는 24시간 뒤에도 사라지지 않습니다
✓ 가족 한 명에게라도 말하면 사기범의 마법이 풀립니다
```

## ✏️ 활동 ① — 사기 단톡방 알아보기 퀴즈

다음 중 사기 신호가 몇 개 보이나요? (정답: 모두)

```
오늘의 추천종목 ▶ ○○바이오
- 단기 +30% 확실
- 정부 정책 수혜주
- 내부 호재 정보 있음
- VIP만 공유
- 오늘 오후 3시 전 매수 필수
- 매수 후 인증샷 필수
```

→ 정답: **6개 모두 사기 신호**. 이 중 단 하나만 보여도 끝.

---

## ✏️ 활동 ② — 거절 대본 외우기

투자 권유 카톡이 올 때 쓸 한 마디를 정해두세요.

```
1. "투자는 가족과 의논해서만 합니다. 감사합니다."

2. "저는 모르는 분과 투자 이야기 안 합니다."

3. (그냥 차단)
```

> 💡 **가장 강력한 답은 "차단"입니다.** 이유 댈 필요 없어요. 무례한 게 아니라 안전한 겁니다.

---

## ✏️ 활동 ③ — 우리 집 투자 규칙 정하기

가족과 함께 정해두는 규칙 (활동북에 적기):

```
□ 100만 원 이상 투자는 ____________ 와 먼저 의논한다
□ 모르는 사람이 권하는 투자는 무조건 거절한다
□ 카톡·문자로 오는 투자 정보는 무조건 무시한다
□ "확실하다"는 말이 나오면 무조건 사기로 간주한다
□ 한 번 잃은 돈은 깔끔하게 포기한다 (회수 시도가 두 번째 함정)
```

---

## 💬 가족과 함께하기

이번 장 내용을 가족 단톡방에 이렇게 공유해보세요.

> *"우리 가족 규칙 하나 더.
> 모르는 사람이 카톡으로 투자 권하면 무조건 거절하기.
> '확실하다'는 말 나오면 100% 사기래.
> 큰돈 움직이기 전에 가족한테 한 번만 말해주기."*

## 🆘 만약 이미 송금하셨다면

```
1. ☎ 112 즉시 신고 (30분 골든타임)
2. 은행 지급정지 신청
3. 카카오톡 대화 캡처 보관 (증거)
4. ☎ 1332 금융감독원 피해구제
5. 절대 추가 송금 금지
   ("회수해드린다"는 2차 사기범도 많습니다)
```

> ⚠️ **가장 위험한 순간은 피해 직후입니다.** 사기범 또는 그 일당이 "변호사", "회수 전문가"를 사칭해 다시 접근합니다. **이미 잃은 돈은 잃은 거**라고 인정하는 것이 두 번째 피해를 막는 길입니다.

## 🛡️ 이번 장 핵심 — 멈춤 5초로 다시 외우기

투자 사기에도 멈춤 5초 루틴이 그대로 통합니다.

| 단계 | 투자 사기 상황에서 |
|---|---|
| 1. **멈춤** | 송금 버튼 누르기 전, 손가락 떼기 |
| 2. **의심** | "이거 진짜 맞나?" 입으로 말하기 |
| 3. **확인** | 가족에게 전화 — "이런 거 권유받았는데 어때?" |
| 4. **끊기** | 단톡방 나가기·차단·답장 안 함 |
| 5. **알리기** | 가족·112·1332 신고 |

> **"멈춰, 의심, 확인, 끊고, 알려."**
> 송금하기 전 5초만 멈추면 평생 모은 돈을 지킵니다.
