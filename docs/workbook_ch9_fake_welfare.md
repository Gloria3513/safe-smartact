# 9장. "어르신만 받을 수 있는 혜택입니다" — 가짜 정부지원금·복지 사기

## 📖 오늘의 한 문장
> **"진짜 정부 혜택은 신청해도 받기 어렵고, 가짜 정부 혜택은 신청 안 해도 알아서 찾아옵니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **정부지원금 사칭** | 보건복지부·국민연금·구청을 사칭해 돈·정보를 빼앗는 사기 |
| **수급자 사칭** | "어르신은 이 혜택 받을 수 있습니다" 하며 접근하는 수법 |
| **신청대행 사기** | "내가 대신 신청해주겠다" 하며 수수료·개인정보 가로채기 |
| **에너지바우처·기초연금** | 진짜 복지제도. 사기범이 가장 많이 사칭하는 이름 |
| **공식 통로** | 동주민센터·복지로(bokjiro.go.kr)·국민연금공단 1355 |

---

## 🎙️ 실제 사례 — 한분례 님(72세) 이야기

어느 오후, 친절한 목소리의 전화가 걸려왔습니다.

> *"안녕하세요, 보건복지부 고령자복지과 김주임입니다. 어머님 이번에 새로 시행되는 '시니어 건강지원금' 80만 원 받으실 수 있어요. 아직 신청 안 하셨더라고요."*

한 씨는 처음 듣는 이야기였지만, **"받을 수 있다"** 는 말에 마음이 흔들렸습니다.

> *"신청은 간단합니다. 휴대폰으로 본인확인만 해주시면 돼요. 지금 문자 한 통 보내드릴 테니, 그 링크 누르시고 주민번호랑 계좌번호만 입력하세요. 5분이면 끝나요."*

한 씨는 시키는 대로 했습니다. 그 다음 날, **통장에 있던 1,800만 원이 모두 빠져나가 있었습니다.** 링크는 가짜였고, 사기범은 **공인인증서와 OTP까지 한 번에 가져갔던 것**입니다.

> 한 씨는 가장 후회했습니다. *"동주민센터에 한 번만 전화해서 물어볼걸…"*

---

## ⚠️ 위험신호 — 이런 말이 나오면 즉시 멈춤

```
□ "보건복지부", "국민연금공단", "구청" 사칭 전화
□ "어르신만 받을 수 있는 혜택"
□ "아직 신청 안 하셨다", "마감 임박"
□ "본인확인만 하면 된다"
□ 문자로 링크를 보내며 주민번호·계좌번호 요구
□ "공인인증서·OTP·보안카드" 입력 요청
□ "1대1 신청대행 해드린다" + 수수료 요구
□ "이번에만 가능한 특별지원금"
□ "동주민센터에 가실 필요 없다"
□ 카카오톡으로 "정부지원금 안내" 친구추가
```

> **이 중 단 하나라도 나오면 100% 사기입니다.** 진짜 정부 기관은 전화로 주민번호·계좌번호 요구하지 않습니다.

---

## 🧠 왜 시니어가 표적이 될까?

### ① "혜택을 놓치면 손해"라는 마음

평생 세금 내며 살아온 어르신들은 **"나도 받을 권리가 있다"** 는 마음이 큽니다. 사기범은 정확히 이 심리를 노립니다.

### ② "진짜 복지제도가 너무 많다"

기초연금·에너지바우처·노인일자리·건강검진… 진짜 정부 혜택이 너무 많아서, **새로운 이름이 나와도 의심하기 어렵습니다.** 사기범은 이 틈을 파고듭니다.

### ③ "공식기관 이름의 권위"

"보건복지부", "국민연금공단" 같은 이름은 의심하기 힘듭니다. 사기범은 이 이름을 그대로 도용합니다.

### ④ "복잡한 절차에 대한 두려움"

복지 신청은 정말로 복잡합니다. "내가 대신 해드릴게요"라는 말이 너무 친절하게 들립니다. 그게 함정입니다.

---

## 🛡️ 절대 흔들리지 않는 4가지 원칙

### 원칙 1 — 정부 혜택은 "전화로 알려주지 않는다"

```
✓ 진짜 정부 혜택은 본인이 직접 찾아 신청해야 받습니다
✓ "받을 수 있는데 신청 안 하셨다" — 100% 사기
✓ 보건복지부·구청은 개인에게 "혜택 권유 전화" 안 합니다
✓ 진짜 안내는 우편이나 동주민센터 방문 안내문으로 옵니다
```

### 원칙 2 — 주민번호·계좌번호·OTP는 "절대 전화로 말하지 않는다"

```
✓ 진짜 공무원은 전화로 주민번호 묻지 않습니다
✓ 진짜 은행도 전화로 계좌번호·비밀번호 묻지 않습니다
✓ OTP·보안카드 번호는 어떤 기관도 묻지 않습니다
✓ 묻는 순간 사기입니다. 예외 없습니다.
```

### 원칙 3 — 모든 의심은 "동주민센터 한 통화"로 해결

```
✓ 동주민센터 ☎ 본인 동네 번호 (네이버 검색)
✓ "방금 이런 전화가 왔는데 진짜인가요?" 한 마디면 끝
✓ 진짜라면 직접 방문해서 신청
✓ 가짜라면 그 자리에서 안심
```

### 원칙 4 — 신청대행은 무조건 거절

```
✓ "내가 대신 신청해드릴게요" — 100% 사기
✓ 진짜 신청은 본인이 직접 동주민센터에 가서 합니다
✓ 어렵다면 가족·동주민센터 직원이 도와줍니다
✓ 외부 사람이 대신해주는 정부지원금은 없습니다
```

## ✏️ 활동 ① — 진짜와 가짜 구별 퀴즈

다음 중 진짜는 어느 것일까요?

```
A. 동주민센터에서 우편으로
   "기초연금 신청 안내문"이 옴

B. "보건복지부입니다. 어머님 받으실 혜택이
   80만원 있는데 본인확인 부탁드립니다"라며
   문자로 링크 보냄
```

→ 정답: **A만 진짜**.
- B는 100% 사기 (정부는 문자 링크로 본인확인 안 함)
