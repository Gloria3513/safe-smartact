# 2장. 택배·과태료 문자의 함정 — 스미싱

## 📖 오늘의 한 문장
> **"문자 속 링크는 누르기 전에 한 번만 멈추면 됩니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **스미싱** | 문자(SMS)로 보낸 가짜 링크로 속이는 사기 |
| **악성앱** | 누르면 몰래 설치되어 문자·연락처를 훔쳐가는 앱 |
| **도메인** | 인터넷 주소의 꼬리 (예: `.kr` `.com` `.top`) |
| **1118** | 스팸·스미싱 문자를 신고하는 전화번호 |

---

## 📱 실제 사례 — 박정호 님(68세) 이야기

박 씨는 아침에 문자 한 통을 받았습니다.

> *[CJ대한통운] 주소불일치로 배송이 보류중입니다. 주소 확인 → http://cj-ko.delivery-r.top/2k9*

"어제 손주 선물 시킨 게 있는데…" 박 씨는 무심코 링크를 눌렀습니다. 이상한 앱 하나가 설치됐습니다. 그날 오후, **박 씨 번호로 가족들에게 똑같은 문자가 쏟아졌습니다.** 연락처가 통째로 넘어간 것입니다.

---

## ⚠️ 위험신호 — 이 문자는 100% 사기입니다

```
□ 택배·과태료·청첩장 + 이상한 링크
□ 주소(도메인)가 .top / .xyz / .click / .vip 로 끝남
□ "확인" "보류" "결제실패" 같은 급한 단어
□ 내가 주문하지도 않은 택배
□ 숫자와 영문이 뒤섞인 이상한 주소
```

> **택배회사는 문자에 링크를 거의 넣지 않습니다.** 의심되면 공식 앱을 직접 열어 확인하세요.

---

## 🛡️ "누르기 전 1분" 체크 3단계

1. **누가 보냈나?** — 번호가 평소와 다른가
2. **도메인이 무엇인가?** — `.kr` / `.com` 이 아니라 `.top` `.xyz`면 의심
3. **내가 지금 이걸 기다리고 있었나?** — 기대 안 한 것이면 일단 멈추기

## ✏️ 활동 ② — 의심 문자 바로 신고하기

```
[방법 1] 문자 길게 누르기 → 신고 → 스팸
[방법 2] 118 전화 → "스미싱 문자 신고합니다"
[방법 3] 가족 단톡방에 캡처 공유해서 주의 알리기
```

---

## 🆘 만약 링크를 눌렀다면

```
1. 비행기모드부터 (통신 차단)
2. 방금 설치된 앱 확인 → 삭제
3. 통신사 고객센터 → "소액결제 차단" 요청
4. 은행앱 로그인 비밀번호 변경
5. ☎ 118 (KISA) 에 신고
```

> 📌 **당황하지 마세요. 빠른 차단이 가장 중요합니다.**
