# 12장. 목소리로 겁주는 사기 — 비싱(보이스피싱)

## 📖 오늘의 한 문장
> **"검찰·경찰은 전화로 계좌번호를 묻지 않습니다. 끊고 112로 다시 거세요."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **비싱(Vishing)** | '목소리(Voice) + 피싱'. 전화로 겁을 줘서 돈이나 계좌를 빼앗는 사기 |
| **검찰 사칭** | "○○지검 ○○검사입니다" 로 시작하는 대표 수법 |
| **대포통장** | 남의 이름으로 만든 통장. 사기꾼이 돈을 받아가는 통로 |
| **112** | 경찰 신고. 보이스피싱 의심 즉시 전화 |

---

## ☎️ 실제 사례 — 최영자 님(70세) 이야기

점심 때 걸려 온 전화. 차분한 남자 목소리였습니다.

> *"안녕하십니까, 서울중앙지검 김민수 검사입니다. 최영자 님 명의로 개설된 통장이 범죄에 사용되어, 지금 계좌의 돈을 안전계좌로 이체하셔야 합니다."*

최 씨는 심장이 내려앉았습니다. 검사가 부른 "안전계좌"로 3,200만 원을 이체했습니다. **안전계좌는 없습니다.** 그날 저녁 진짜 검찰청에 전화해 확인했을 땐 이미 돈이 해외로 빠져나간 뒤였습니다.

---

## ⚠️ 위험신호 — 이 전화는 100% 사기입니다

```
□ "검찰" "경찰" "금감원" 이라고 스스로 말함
□ "안전계좌로 이체" 라는 말이 나옴
□ "전화를 끊지 마세요" "주변에 말하지 마세요"
□ 숫자/앱 설치를 유도 (원격제어 앱 포함)
□ 갑자기 급하게 돈을 옮기라고 재촉
```

> **검찰·경찰·금감원은 전화로 돈을 옮기라고 하지 않습니다.** 100% 사기입니다.

---

## 🛡️ "전화 끊기 전 1분" 체크 3단계

1. **"검찰" "경찰" "금감원" 말했는가?** — 말한 순간 99% 사기
2. **"안전계좌" "이체" 말했는가?** — 이 단어가 나오면 100% 사기
3. **주변에 말하지 말라고 했는가?** — 입막음은 사기꾼의 특징

**→ 하나라도 해당되면 바로 끊고, 가족이나 112에 전화하세요.**

---

## ✏️ 활동 ① — 이 전화 진짜/가짜 가려내기

| 받은 전화 내용 | 진짜 / 가짜 |
|---|---|
| "국세청입니다. 세금 문의는 126번으로 다시 걸어주세요" (길게 설명 없음) | ☐ |
| "서울중앙지검 검사입니다. 지금 안전계좌로 이체하세요" | ☐ |
| "아들입니다. 제가 사고가 나서 급히 돈이 필요해요. 아빠한테는 말하지 마세요" | ☐ |
| "병원 예약 확인 전화드립니다. 내일 10시 맞으시죠?" | ☐ |
| "금감원입니다. 고객님 계좌에 범죄자금이 들어와 지금 옮기셔야 합니다" | ☐ |

> *정답: 진짜 / 가짜 / 가짜(메신저피싱 포함) / 진짜 / 가짜*
> 핵심: **"검찰/경찰/금감원 + 이체/안전계좌/비밀유지"** = 100% 사기

---

## ✏️ 활동 ② — 전화 받을 때 3초 멈춤 연습

```
[1단계] 떨리는 손으로 전화 받기
[2단계] "검찰" "이체" "말하지 마세요" 들리면 → 3초 멈춤
[3단계] "끊고 다시 걸겠습니다" 하고 끊기
       → 112 또는 가족에게 전화
```

> 💡 **진짜 기관이면 다시 전화해도 연결됩니다. 가짜는 재연결이 안 돼요.**

---

## 🆘 만약 이체했다면 — 30분 안에!

```
1. 거래 은행 고객센터 즉시 전화
   → "보이스피싱 피해, 지급정지 요청합니다"
2. 경찰청 ☎ 112 신고
3. 금융감독원 ☎ 1332 — 피해구제 신청
4. 입금 직후 30분 안이면 "사기이용계좌 지급정지"로 되찾을 확률 큼
5. 피해 금액·통화 시각·상대 번호 기록 남기기
```

> 📌 **지급정지는 빠를수록 돈을 찾을 확률이 높아집니다. 부끄러워 마시고 바로 전화!**
