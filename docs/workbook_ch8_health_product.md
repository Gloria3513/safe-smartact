# 8장. "이것만 드시면 다 낫는다" — 건강식품·홈쇼핑 과장광고

## 📖 오늘의 한 문장
> **"진짜 약은 약국에 있고, 진짜 의사는 병원에 있습니다. TV 속에는 없습니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **건강기능식품** | 식약처가 일부 기능을 인정한 식품. **약이 아닙니다.** |
| **일반식품** | 그냥 음식. 어떤 효능도 광고할 수 없습니다. |
| **체험단·간증** | "저도 이거 먹고 다 나았어요" — 대부분 대본·연기자 |
| **닥터쇼핑** | 의사 가운 입은 사람이 TV·유튜브에서 특정 제품 추천하는 것 |
| **자동결제** | 한 번 주문하면 매달 자동으로 돈이 빠져나가는 함정 |

---

## 🎙️ 실제 사례 — 최복순 님(74세) 이야기

무릎이 아파 고생하던 어느 날, TV 홈쇼핑에서 **흰 가운을 입은 의사**가 나왔습니다.

> *"이 ○○관절 영양제는 90% 이상이 3개월 안에 효과를 봅니다. 오늘 방송에서만 30% 할인! 매진 임박!"*

옆에서 **할머니 출연자**가 눈물을 흘리며 말했습니다.

> *"저는 10년 된 무릎 통증이 한 달 만에 싹 사라졌어요. 이제 손주랑 등산도 가요."*

최 씨는 9만 9천 원짜리 3개월치를 주문했습니다. 한 달 뒤, 통증은 그대로였습니다. 그런데 통장에서는 매달 9만 9천 원이 빠져나가고 있었습니다. **자동결제**였습니다.

전화해서 해지하려 하자 ARS는 30분 대기, 상담원은 "본인이 동의하셨다"고만 반복했습니다. 6개월 동안 약 60만 원이 빠져나간 뒤에야 해지에 성공했습니다.

> 더 큰 문제는, 그 사이에 진짜 무릎 치료를 받지 않아 **연골이 더 닳았다**는 것이었습니다.

---

## ⚠️ 위험신호 — 이런 광고는 무조건 의심

```
□ "○○병이 깨끗하게 낫는다"
□ "약을 끊을 수 있다"
□ "수술 안 해도 된다"
□ "의사가 추천한다" + 흰 가운
□ "체험단 90%가 효과 봤다"
□ "오늘만 할인", "매진 임박", "한정 수량"
□ "전화 한 통이면 끝", "ARS만 누르세요"
□ "자동배송 등록 시 추가 할인"
□ 유튜브 광고 + 댓글 모두 "효과 봤어요"
□ 카카오톡으로 친구·이웃이 추천 (다단계 신호)
```

> **2개 이상 보이면 99% 과장광고입니다.** 진짜 약이라면 의사 처방이 필요합니다.

---

## 🧠 왜 자꾸 속을까? — 시니어를 노리는 함정 4가지

### ① 흰 가운의 권위

TV에 의사 가운을 입고 나오는 사람의 80%는 **광고 출연료를 받은 사람**입니다. 진짜 의사일 수도 있고, 단순 모델일 수도 있습니다. **흰 가운이 곧 신뢰는 아닙니다.**

### ② 또래 출연자의 공감

비슷한 연배의 어르신이 "저도 이거 먹고 좋아졌어요" 하면 마음이 흔들립니다. 그 어르신은 **대본을 받고 출연료를 받은 분**입니다. 진짜 사용자가 아닙니다.

### ③ "오늘만"의 긴급함

"오늘 방송에서만", "매진 임박" — 이 말은 멈춤을 막기 위한 장치입니다. 진짜 좋은 제품은 오늘 안 사도 다음 주에 살 수 있습니다.

### ④ 자동결제의 함정

처음엔 "1회 체험가 9,900원"이라고 하지만, 작은 글씨로 **"체험 후 자동으로 정기 배송"**이 적혀 있습니다. 시니어가 가장 많이 당하는 함정입니다.

## 🛡️ 4가지 황금 규칙

### 규칙 1 — TV·유튜브 광고로 약 사지 않기

```
✓ 진짜 약은 약사·의사를 통해서만 삽니다
✓ 건강기능식품은 마트·약국에서도 살 수 있습니다
✓ TV 광고 가격이 더 비싼 경우가 많습니다
```

### 규칙 2 — "전화 주문" 대신 "직접 약국·병원" 가기

```
✓ 무릎이 아프면 정형외과로
✓ 혈압이 높으면 내과로
✓ 광고 제품 사기 전에 의사·약사에게 한 번만 물어보기
✓ "이거 어때요?" 한 마디면 충분
```

### 규칙 3 — 자동결제는 무조건 거절하기

```
✓ "정기배송" 들어간 상품은 모두 거부
✓ 첫 주문 시 "단 한 번만 결제"라고 명시 요청
✓ 의심되면 한 달 뒤 통장 꼭 확인하기
```

### 규칙 4 — 가족과 의논 후 결정

```
✓ "이거 사도 될까?" 자녀·손주에게 한 마디
✓ 광고 화면 캡처해서 가족 단톡방에 보내기
✓ 가족이 검색해주면 진짜인지 1분 만에 확인 가능
```

## ✏️ 활동 ① — 광고 문구 판별 퀴즈

다음 문구 중 사기·과장 신호는 몇 개일까요? (정답: 모두)

```
✓ "이거 한 통이면 당뇨가 깨끗이 낫습니다"
✓ "10만 명이 효과를 본 ○○제품"
✓ "오늘 방송에서만 50% 할인"
✓ "약을 끊으셔도 됩니다"
✓ "의사가 직접 추천한 ○○성분"
✓ "정기 배송 신청 시 추가 30% 할인"
```

→ 정답: **6개 모두 위험 신호**.
- "낫는다"는 식품광고법 위반
- "약을 끊는다"는 의약품 광고 금지 표현
- "정기 배송"은 자동결제 함정

---

## ✏️ 활동 ② — 가족과 함께 통장 점검하기

이번 달 통장에서 다음 항목을 확인해보세요.

```
□ 매달 똑같이 빠져나가는 작은 금액 (5,000원~30,000원)
□ 모르는 회사 이름의 출금 (○○헬스, ○○라이프, ○○샵 등)
□ "정기결제" 또는 "월정액"이 표시된 출금
```

해당되는 게 있다면, **자녀와 함께 카드사·은행에 전화**해서 바로 해지하세요. 부끄러워하지 마세요. 매우 흔한 일입니다.

---

## ✏️ 활동 ③ — 거절 대본 외우기

홈쇼핑·전화 영업 거절 대본 3종.

```
1. "필요 없습니다. 끊겠습니다."
   (이유 댈 필요 없음)

2. "병원에서 처방받은 약만 먹습니다. 감사합니다."

3. "가족과 의논하고 다시 연락드릴게요."
   (사실은 안 연락함)
```

> 1분이면 진짜인지 확인할 수 있어.

## 🆘 만약 이미 결제하셨다면

```
1. 카드사 ☎ 1599 또는 1588 전화
   → "정기결제 해지 + 청약철회" 요청
2. 7일 이내라면 무조건 환불 가능 (전자상거래법)
3. 이미 7일 지났어도 자동결제는 즉시 해지 가능
4. 한국소비자원 ☎ 1372 (피해상담)
5. 부끄러워하지 말기 — 매우 흔한 일입니다
```

> 💡 **"청약철회"** 이 한 마디만 알면 7일 안에는 무조건 환불 가능합니다. 카드사·쇼핑몰이 거절하면 1372에 신고하세요.

## 🛡️ 이번 장 핵심 — 멈춤 5초로 다시 외우기

홈쇼핑·과장광고에도 멈춤 5초가 그대로 적용됩니다.

| 단계 | 광고 시청 상황에서 |
|---|---|
| 1. **멈춤** | "전화 주문" 누르기 전, 리모컨 내려놓기 |
| 2. **의심** | "정말 그렇게 좋다고?" 입으로 말하기 |

| 4. **끊기** | TV 채널 돌리기 / 전화 영업 그냥 끊기 |
| 5. **알리기** | 자동결제 발견하면 가족에게 즉시 공유 |

> **"멈춰, 의심, 확인, 끊고, 알려."**
