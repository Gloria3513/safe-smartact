# 활동북 인쇄 조판 준비 가이드 — *속지 않는 하루*

1부 5장(보이스피싱·스미싱·메신저피싱·로맨스스캠·AI음성복제)을 실제 인쇄 가능한 책으로 만드는 작업 가이드. Figma 작업은 사용자가 직접, 콘텐츠는 이미 완성.

---

## 📏 책 사양

| 항목 | 값 |
|---|---|
| 판형 | A5 (148×210mm) — 시니어가 손에 잡기 편함 |
| 분량 | 본문 80p + 표지·판권·목차 12p = 총 92p |
| 종이 | 본문 미색모조 100g / 표지 스노우 250g 무광코팅 |
| 제본 | 무선제본 (풀제본) |
| 인쇄 | 본문 1도 흑백 / 표지 4도 컬러 |
| 활동북용 특성 | 쓰기 공간 넉넉, 글씨 크기 14pt 이상 |

---

## 🎨 디자인 원칙 (시니어 활동북)

### 타이포그래피
- **본문:** Pretendard 14pt, 행간 1.7
- **소제목:** Pretendard Bold 18pt
- **큰 제목:** Pretendard ExtraBold 24pt+
- **쓰기 칸:** 최소 20mm 높이 (손으로 쓰기 충분)

### 색상 (흑백이지만 강조는 회색 톤으로)
```
본문:       #1a1a1a
강조:       #000000 (Bold)
보조설명:   #5a5a5a
배경강조:   #f0f0f0 (카드)
쓰기칸:     #ffffff (흰색) + 점선 테두리
```

### 일러스트
- 손그림 스타일 권장 (AI 이미지로도 OK)
- 시니어가 등장하는 장면 (식탁에서 문자 보는 할아버지, 카톡 보는 할머니 등)
- 과한 경고 이미지 금지 (공포 유발 NO, 친근한 톤 YES)

## 🖼️ Figma 조판 순서 (하루 작업)

### Step 1. 파일 생성 (10분)
1. Figma에서 **새 파일** 생성 → 이름: `속지않는하루_v1`
2. 페이지 구조:
   - **Cover** (표지·뒷표지)
   - **Front Matter** (판권·인사말·목차)
   - **Ch 1** ~ **Ch 5** (본문 5개 장)
   - **Back Matter** (부록)
3. 프레임 크기: **148 × 210mm** (A5)

### Step 2. 마스터 페이지 만들기 (30분)
- 머리말: 장 번호 + 제목
- 꼬리말: 쪽번호 + "속지 않는 하루 · stopcall.kr"
- 여백: 상·하 18mm, 좌·우 16mm, 안쪽 여백 20mm (제본 고려)

### Step 3. 장 템플릿 (1장 기준 2시간)

각 장은 12페이지 = 6장 전개. `workbook_ch1_voice.md` 파일 내용을 페이지별로 분할:

```
p1    장 도비라(큰 제목 + 일러스트)
p2    📖 오늘의 한 문장 + 🔍 용어 카드
p3-4  📞 실제 사례 (김순자 님 이야기) + 일러스트
p5    ⚠️ 위험신호 체크박스
p6    🛡️ 핵심 대응법 3문장
p7    ✏️ 활동 ① 따라 써보기 (쓰기 공간)
p8    ✏️ 활동 ② ○× 점검
p9    🆘 응급 대응 순서
p10   💬 가족과 함께하기
p11   📝 오늘의 한 줄 일기 (쓰기 공간)
p12   QR + 다음 장 안내
```

### Step 4. 다른 장으로 복제 (2-5장 각 30분)
1장 템플릿을 복제해서 각 장 콘텐츠로 교체. `docs/workbook_ch2_smishing.md`, `ch3_messenger.md`, `ch4_romance.md`, `ch5_ai_voice.md` 그대로 사용.

### Step 5. 표지 디자인 (1시간)
- 메인 카피: **속지 않는 하루**
- 서브: *시니어 사이버안전 활동북*
- 일러스트: 할머니가 전화기를 들고 있는 장면 (손 흔들며 "끊을게요!")
- 뒷표지: QR + 스마택트 로고 + 발행정보

### Step 6. 인쇄용 내보내기 (30분)
1. Figma → **Export as PDF**
2. 옵션:
   - Scale: 1x
   - Include bleed: ✅ (3mm)
   - Color profile: CMYK (인쇄용)
3. 파일명: `속지않는하루_인쇄용_v1.pdf`

## 🖨️ 인쇄 주문 (와우프레스)

와우프레스는 이미 스마택트 인쇄 라인으로 쓰고 있어 계정·API 연동이 되어 있을 것.

### Step 1. 1권 샘플 인쇄 (미팅 들고 갈 용)
- 목적: 실물 품질 검증 + 영업 시연용
- 수량: **1권**
- 비용: 약 1~2만원 (샘플가)
- 소요: 3~5일

### Step 2. 품질 점검
- 글자 크기가 A5에서 실제로 읽히는지
- 쓰기 칸이 손 크기에 맞는지
- QR이 실제로 스캔되는지
- 제본이 펼쳐서 쓸 수 있는지 (활동북 핵심)

### Step 3. 초도 본인쇄
- 첫 주문: **30~50권** (파일럿 복지관용)
- 수량 늘수록 권당 단가 하락
- 500권 기준 권당 3,000~5,000원

## 🎯 오늘 하루 작업으로 끝낼 수 있는 범위

**현실적으로 하루 집중하면:**
- ✅ Figma 파일 생성 + 마스터 페이지
- ✅ 1장(보이스피싱) 전체 조판 완료
- ✅ 표지 시안 1종
- ⏳ 2~5장은 1장 템플릿 복제 후 내용 교체 (1장 완성 후엔 속도 빠름)

**분량 걱정된다면:**
1부 5장 전체를 한꺼번에 찍지 말고, **1장만 먼저 32p 소책자(A5)** 로 뽑아서 샘플 제작. 미팅 반응 보고 나머지 진행.
