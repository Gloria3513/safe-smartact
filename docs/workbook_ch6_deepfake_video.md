# 6장. 화면 속 얼굴마저 가짜인 시대 — 딥페이크 영상통화

## 📖 오늘의 한 문장
> **"눈으로 봤다고 믿지 마세요. 화면 속 얼굴은 더 이상 증거가 아닙니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **딥페이크 영상통화** | 가족·지인의 얼굴을 AI로 본떠 실시간 영상통화로 거는 사기 |
| **실시간 페이스스왑** | 사기범의 얼굴 위에 다른 사람 얼굴을 덮어씌우는 기술 |
| **물리 검증** | 화면 속 사람에게 손짓·물건 들기를 시켜 진짜인지 확인하는 방법 |
| **콜백 확인** | 영상통화를 끊고 원래 알던 번호로 다시 거는 것 |

## 🎙️ 실제 사례 — 박순희 님(68세) 이야기

저녁 8시쯤, 카카오톡 영상통화가 울렸습니다. 화면에 **딸 얼굴**이 떴습니다.

> *"엄마, 나야. 회사에서 큰 사고가 났어. 거래처 돈 1,200만 원을 잘못 보냈는데, 내일 아침까지 못 채우면 해고야. 엄마, 도와줘…"*

화면 속 딸은 울고 있었고, 입술도 움직이고 있었습니다. 박 씨는 의심할 여지가 없었습니다. 30분 안에 1,200만 원을 송금했습니다.

다음 날 아침, 출근한 딸은 멀쩡했습니다. **사기범은 딸의 인스타그램 사진 5장과 짧은 릴스 영상 하나로 실시간 딥페이크 영상통화를 만들어낸 것**입니다.

> 박 씨가 나중에 가장 후회한 것은 "엄마, 손 흔들어봐"라고 한 번만 말해봤더라면 — 이었습니다.

---

## ⚠️ 위험신호 — 영상통화에서 의심해야 할 순간

```
□ 가족·지인인데 발신번호가 평소와 다름
□ 화면 속 얼굴이 거의 움직이지 않음 (정면 고정)
□ 눈 깜빡임이 너무 적거나, 너무 일정함
□ 입모양과 목소리가 살짝 어긋남 (싱크 0.3초 차이)
□ 머리카락·귀·턱선 경계가 흐릿하게 번짐
□ 손을 화면에 보여주지 않음 (얼굴만 클로즈업)
□ 통화가 짧고, 더 보지 못하게 서두름
□ "급한 돈" + "아무한테도 말하지 마"
```

> **2026년 현재, 인스타·유튜브 사진 몇 장만 있으면** 누구나 실시간 딥페이크 영상통화를 만들 수 있습니다. 손주가 SNS에 부모님·조부모님 사진을 올렸다면, 누구든 표적이 될 수 있습니다.

## 🛡️ 물리 검증 3종 세트 — 딥페이크가 못 따라하는 것

딥페이크 AI는 **얼굴 정면**만 진짜처럼 만들어냅니다. 화면 밖으로 손을 뻗거나, 물건을 가져오거나, 옆모습을 보이는 것은 어색해집니다.

### ① 손 흔들기 검증

> *"잠깐, 손 한 번 흔들어볼래?"*

진짜 딸: 자연스럽게 손을 흔듦
딥페이크: 손이 안 보이거나, 손과 얼굴이 겹치면 화면이 일그러짐

### ② 옆모습 검증

> *"고개 한 번 옆으로 돌려봐. 90도로."*

진짜 딸: 옆얼굴이 자연스럽게 보임
딥페이크: 옆모습이 깨지거나, 얼굴이 흐려짐

### ③ 물건 검증 (가장 강력)

> *"옆에 있는 책이나 물병 하나 집어서 화면에 보여줄래?"*

진짜 딸: 자연스럽게 물건을 집어 보여줌
딥페이크: "지금 손이 묶여있어" / "보여줄 수가 없어" / 화면이 끊김

> 💡 **이 세 가지 중 하나만 시켜봐도 충분합니다.** 사기범이 가장 두려워하는 요청입니다.

---

## 🗝️ 가족 본인확인 암호 — 영상통화에서 두 배 강력

1장에서 정한 **본인확인 암호**가 여기서도 핵심입니다.

> *"잠깐, 우리 집 암호 한 번 말해볼래?"*

딥페이크는 얼굴은 복제해도 **암호는 모릅니다.** 진짜 자식은 웃으면서 답하고, 사기범은 말을 돌립니다 — *"엄마 지금 그게 중요해?"*

이 한 마디가 나오면 100% 사기입니다.

## ✏️ 활동 ① — 검증 대본 외우기

아래 세 문장을 큰 소리로 세 번 읽어보세요. 머리가 아니라 입이 외워야 합니다.

```
1. "잠깐, 손 한 번 흔들어볼래?"

2. "옆에 있는 물건 하나 집어서 보여줘봐."

3. "급한 거 알아. 우리 집 암호 한 번 말해줘."
```

이 세 문장 중 **하나라도** 자연스럽게 못 하면 → 즉시 끊고 콜백.

**얼굴+영상 둘 다 공개된 사람이 있다면**, 그 사람은 딥페이크 영상통화 사칭의 재료가 됩니다. 가족 단톡방에 이 사실을 공유하세요.

> 📌 SNS를 다 비공개로 바꾸지 않아도 됩니다. 대신 **"우리 가족은 영상통화로 돈 얘기 안 한다"** 라는 규칙 하나만 정해두세요.

## 🧠 한 걸음 더 — 딥페이크가 못 만드는 4가지

```
1. 빠른 옆얼굴 회전 (90도 이상)
2. 손이 얼굴을 가로지르는 동작
3. 입에 물건을 무는 동작 (펜·과일)
4. 갑자기 일어서서 전신 보여주기
```

영상통화로 의심스러운 부탁을 받았을 때, 이 4가지 중 **2가지 이상**을 시켜보세요. 진짜 가족은 웃으며 다 합니다.


## 🛡️ 이번 장 핵심 — 멈춤 5초로 다시 외우기

영상통화 사기에도 멈춤 5초 루틴은 똑같이 적용됩니다.

| 단계 | 영상통화 상황에서 |
|---|---|
| 1. **멈춤** | 화면에서 시선 한 번 떼고 숨 고르기 |
| 2. **의심** | "이거 이상한데?" 입으로 말하기 |
| 3. **확인** | 손 흔들기·물건 보여주기·암호 물어보기 |
| 4. **끊기** | 어색한 답이 나오면 즉시 끊기 |
| 5. **알리기** | 가족에게 콜백, 112 또는 118 신고 |

> **"멈춰, 의심, 확인, 끊고, 알려."**
> 손가락 5개로 외운 그 챈트, 영상통화에서도 똑같이 통합니다.
