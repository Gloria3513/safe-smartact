# 10장. 이메일로 날아오는 낚시 — 피싱

## 📖 오늘의 한 문장
> **"낚시꾼은 이메일 제목에 '긴급'을 씁니다."**

---

## 🔍 용어 카드

| 용어 | 뜻 |
|---|---|
| **피싱(Phishing)** | 영어로 '낚시(fishing)'에서 온 말. 이메일로 속여서 비밀번호·돈을 낚는 사기 |
| **가짜 로그인 화면** | 네이버·국민은행·카카오처럼 보이지만 주소가 다른 속임수 페이지 |
| **첨부파일** | 이메일에 붙은 문서·그림. 열면 바이러스가 깔릴 수 있어요 |
| **인터넷진흥원(KISA) 118** | 피싱 이메일 신고 전화번호 |

---

## 📧 실제 사례 — 김순자 님(71세) 이야기

김 씨는 아침 메일함을 열었습니다.

> *[네이버 고객센터] 장기 미접속으로 계정이 24시간 후 삭제됩니다. 아래 링크로 로그인하여 보호해주세요.*

화들짝 놀란 김 씨는 링크를 눌렀습니다. 네이버와 똑같이 생긴 화면에 아이디와 비밀번호를 넣었습니다. 그런데 주소를 자세히 보니 `naver.com`이 아니라 `naver-safe.com`이었습니다. **다음 날 김 씨 네이버 계정이 해커에게 넘어갔고, 친구들에게 돈을 빌려달라는 메일이 뿌려졌습니다.**

---

## ⚠️ 위험신호 — 이 이메일은 100% 사기입니다

```
□ "계정이 24시간 후 삭제됩니다" 같은 긴박한 말투
□ 보낸 사람 주소가 이상함 (예: service@naver-help.co.kr)
□ 로그인·비밀번호·주민번호를 메일에서 요구
□ 제목에 "긴급" "중요" "경고" "미확인"
□ 첨부파일이 .zip .exe 로 끝남
```

> **진짜 네이버·국민은행은 이메일로 비밀번호를 묻지 않습니다.** 계정이 궁금하면 앱을 직접 열어 확인하세요.

---

## 🛡️ "열기 전 1분" 체크 3단계

1. **보낸 사람이 누구인가?** — `@naver.com` 이 맞는지, 아니면 `@naver-help.co.kr` 같은 비슷한 가짜인지
2. **내가 지금 이걸 기다렸나?** — 뜬금없는 "계정 확인" 메일은 99% 가짜
3. **링크 위에 마우스를 올려보기** — 진짜 주소가 뜹니다. `.top` `.xyz` 이면 바로 닫기

---

## ✏️ 활동 ① — 진짜/가짜 이메일 가려내기

| 받은 메일 제목과 보낸이 | 진짜 / 가짜 |
|---|---|
| [국민은행] 이번 달 거래내역 안내 (from noreply@kbstar.com) | ☐ |
| [네이버] 계정 정지 24시간 전, 지금 로그인 (from help@naver-safe.top) | ☐ |
| [카카오] 2단계 인증 코드 (내가 방금 요청함) | ☐ |
| [우체국] 배송 주소 미확인, 즉시 결제 (from post@korea-post.click) | ☐ |
| [국세청] 연말정산 간소화 서비스 오픈 안내 (from webmaster@nts.go.kr) | ☐ |

> *정답: 진짜 / 가짜 / 진짜(내가 요청한 경우) / 가짜 / 진짜*
> 핵심: **보낸 사람 주소의 `@` 뒷부분**이 공식 주소와 정확히 같은지 보세요.

---

## ✏️ 활동 ② — 의심 이메일 바로 신고하기

```
[방법 1] 이메일 열지 말고 "스팸 신고" 버튼 누르기
[방법 2] ☎ 118 (KISA 피싱 신고)
[방법 3] 가족 단톡방에 제목·보낸이 캡처 공유
```

---

## 🆘 만약 로그인 정보를 입력했다면

```
1. 해당 서비스(네이버·카카오·은행) 비밀번호 즉시 변경
2. 같은 비밀번호를 쓴 다른 사이트도 모두 변경
3. 2단계 인증(OTP) 설정
4. 은행·카드사 고객센터에 신고 (카드 재발급 검토)
5. ☎ 118 (KISA) 에 신고
```

> 📌 **첫 30분이 가장 중요합니다. 비밀번호 바꾸기가 최우선.**
