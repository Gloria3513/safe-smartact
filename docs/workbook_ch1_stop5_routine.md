# 1장. 멈춤 5초 — 모든 안전의 출발점

> 활동북 《속지 않는 하루》 1장
> 대상: 만 60세 이상 / 학습시간: 50분 / 강의 + 활동

## 학습 목표

- **5초 멈춤 루틴의 5단계**를 손가락으로 외울 수 있다
- 멈춤이 필요한 **3가지 미끼 신호**(가족·권위·긴급)를 알아챌 수 있다
- 사기 전화·문자·카톡 상황에서 **5단계를 직접 적용**해볼 수 있다
- 가족과 함께 쓸 **본인확인 암호**를 정할 수 있다

---

## 1교시. 왜 하필 5초일까? (10분)

### 순자 어머니 이야기

순자 어머니는 어느 봄날 점심 직후 전화를 받았습니다.

> "어머님, 아드님이 큰 사고가 났습니다. 지금 합의금 200만 원이 필요합니다.
> 늦으면 형사처벌 됩니다. 다른 사람한테 말하지 마세요."

순자 어머니는 가슴이 철렁 내려앉아 송금 버튼을 누르려던 순간—
**딱 5초만** 멈췄습니다.
그리고 아들에게 직접 전화를 걸었더니, 아들은 회사에서 점심을 먹고 있었습니다.

**5초가 200만 원을 지켰습니다.**

### 함께 생각해보기

- 순자 어머니가 멈추지 않았다면 어떻게 됐을까요?
- 여러분도 비슷한 전화를 받아본 적이 있나요?
- 그때 어떤 마음이 들었나요?

## 2교시. 멈춤 5단계 배우기 (10분)

### 5단계 표

| 단계 | 한 마디 | 행동 | 손가락 |
|---|---|---|---|
| 1️⃣ **멈춤** | "잠깐만요" | 손가락 떼고 숨 한 번 | 엄지 |
| 2️⃣ **의심** | "이거 이상한데?" | 입으로 소리내어 말하기 | 검지 |
| 3️⃣ **확인** | "다른 데서 확인할게요" | 가족·공식 앱으로 확인 | 중지 |
| 4️⃣ **끊기** | (말없이) | 변명 없이 끊기 | 약지 |
| 5️⃣ **알리기** | "가족에게 알릴게요" | 가족·112·118 | 새끼 |

### 한 마디 챈트

> **"멈춰, 의심, 확인, 끊고, 알려."**

다 같이 세 번 따라해 봅시다. 손가락 5개를 펴고 하나씩 접으면서.

```
1번    2번    3번
멈춰   멈춰   멈춰
의심   의심   의심
확인   확인   확인
끊고   끊고   끊고
알려   알려   알려
```

## 3교시. 손동작 외우기 (5분)

### 짝꿍과 마주 보고

1. 오른손을 펴서 짝꿍에게 보여줍니다
2. "멈춰" 하며 엄지를 접습니다
3. "의심" 하며 검지를 접습니다
4. "확인" 하며 중지를 접습니다
5. "끊고" 하며 약지를 접습니다
6. "알려" 하며 새끼를 접습니다

세 번 반복.

### 거울 보며 혼자

집에 가서 거울 앞에서 5번. 머릿속이 아니라 **손이 먼저** 외워야 합니다.

## 4교시. 멈춤이 필요한 3가지 신호 (5분)

다음 단어가 나오면 **무조건 멈춤**:

### 🚨 신호 ① — 가족이 위험하대요 (감정 미끼)

> "아드님이 사고가 났어요"
> "따님이 체포됐어요"
> "엄마, 나 폰 고장났어"

### 🚨 신호 ② — 처벌받는대요 (공포 미끼)

> "수사 중입니다"
> "체납되셨습니다"
> "검찰입니다"

### 🚨 신호 ③ — 지금 당장이래요 (긴급 미끼)

> "1시간 안에 안 하면"
> "지금 즉시"
> "오늘 마감"

이 세 단어 중 **하나라도** 나오면 → **멈춤 1단계 진입.**

- **어르신**: (5단계 적용)

### 대본 B — 자녀사칭 카톡

- **사기범**: "엄마, 나야. 폰이 고장나서 새 번호야. 100만 원만 빌려줘…"
- **어르신**: (5단계 적용)

### 대본 C — 택배문자

## 6교시. 본인확인 암호 정하기 (5분)

### 가족과 미리 정해두는 한 마디

진짜 가족인지 확인할 때 쓰는 **우리 집만의 암호**를 정합니다.

좋은 암호:
- 가족만 아는 옛날 기억 ("우리 강아지 이름이 뭐였더라?")
- 일상 대화로 자연스럽게 물을 수 있는 것
- AI나 사기범이 절대 모를 것

나쁜 암호:
- 생일·주소처럼 어디서나 알 수 있는 것
- 너무 어려워서 본인도 까먹는 것

## 오늘의 정리

- 5초 멈춤만으로 사기를 막을 수 있습니다
- 5단계: **멈춰, 의심, 확인, 끊고, 알려**
- 손가락 5개로 외웁니다
- 가족과 본인확인 암호를 정합니다
- **속는 건 어르신의 잘못이 아닙니다. 사기범이 너무 정교한 겁니다.**
