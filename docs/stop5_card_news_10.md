# 멈춤 5초 — 카드뉴스 10장 (SNS 배포용)

> 카카오톡·인스타그램·페이스북 시리즈 게시용
> 디자이너 시안 의뢰용 마스터 텍스트
> 사이즈: 1080×1080 정사각

---

## 1번 카드 — 표지

```
┌─────────────────────────┐
│                         │
│   5초가                 │
│   평생을                │
│   지킵니다              │
│                         │
│   ─────────             │
│   멈춤 5초 루틴          │
│                         │
│   stopcall.kr           │
└─────────────────────────┘
```

- 빨강 배경, 흰 글씨
- 큰 손바닥 일러스트 또는 손가락 5개 아이콘
- 해시태그: #멈춤5초 #스마택트 #시니어사이버안전

---

## 2번 카드 — 순자 어머니 이야기 (요약)

```
어느 봄날 점심,
순자 어머니는 전화를 받았습니다.

"어머님, 아드님이 사고가 났어요.
지금 합의금 200만 원이 필요합니다."

송금 버튼을 누르려던 순간—
딱 5초만 멈췄습니다.

아들은 회사에서 점심을 먹고 있었습니다.

5초가 200만 원을 지켰습니다.
```

- 따뜻한 베이지 배경
- 일러스트: 어머니가 휴대폰을 들고 망설이는 장면

---

## 3번 카드 — 5단계 한눈에

```
멈춤 5초 5단계

1️⃣  멈춰   →  엄지   (1초)
2️⃣  의심   →  검지   (1초)
3️⃣  확인   →  중지   (1~3초)
4️⃣  끊고   →  약지   (즉시)
5️⃣  알려   →  새끼   (1분)

손가락 다섯 개로 외워요.
```

- 5색띠로 단계별 색 구분
- 손가락 일러스트 5컷

---

## 4번 카드 — 1단계 멈춤

```
1️⃣ 멈춤

"잠깐만요."

손가락을 화면에서 떼고
숨 한 번 들이쉬세요.

1초면 충분합니다.
이 1초가 모든 것을 바꿉니다.
```

- 빨강 컬러띠
- 손바닥 일러스트

---

## 5번 카드 — 2단계 의심

```
2️⃣ 의심

"이거 이상한데?"

머릿속이 아니라
입으로 소리내어 말하세요.

말하는 순간
사기범의 마법이 풀립니다.
```

- 노랑 컬러띠
- 입에서 말풍선 나오는 일러스트

---

## 6번 카드 — 3단계 확인

```
3️⃣ 확인

"제가 직접 확인할게요."

⚠️ 걸려온 번호로
   다시 걸지 마세요.

✅ 가족에게 전화하거나
   공식 앱에서 확인하세요.
```

- 파랑 컬러띠
- 두 갈래 화살표 (X / O)

---

## 7번 카드 — 4단계 끊기

```
4️⃣ 끊기

(말없이)

이유 댈 필요 없어요.
변명할 필요도 없어요.
미안해할 필요도 없어요.

그냥 끊으세요.
끊는 건 무례한 게 아닙니다.
```

- 초록 컬러띠
- 통화 종료 버튼 일러스트

---

## 8번 카드 — 5단계 알리기

```
5️⃣ 알리기

"가족에게 알릴게요."

혼자 끌어안지 마세요.
부끄러워하지 마세요.

🆘 가족 단톡방
🆘 112 (경찰)
🆘 118 (KISA 사이버범죄)
🆘 1332 (금융감독원)
```

- 보라 컬러띠
- 손에 손 잡은 일러스트

---

## 9번 카드 — 멈춤이 필요한 3가지 신호

```
🚨 이 단어가 나오면 무조건 멈춤

① "가족이 사고났어요"
   → 감정 미끼

② "처벌받습니다"
   → 공포 미끼

③ "지금 당장이에요"
   → 긴급 미끼

이 셋 중 하나만 나와도
멈춤 1단계 진입.
```

- 검정 배경, 빨강 경고색
- 경고 아이콘

## 10번 카드 — 마지막 메시지

```
잘못한 게 아닙니다.

사기범이
너무 정교한 거예요.

멈출 줄 아는 어르신이
가장 강한 어르신입니다.

──────────
멈춰, 의심, 확인, 끊고, 알려.
──────────

오늘 가족에게
이 카드 보내주세요.

stopcall.kr
멈춤연습
```

- 따뜻한 베이지
- 가족 일러스트 (손주가 할머니 손 잡고)
- 공유하기 유도

## 디자이너 노트

- 폰트: Pretendard 또는 Noto Sans KR Bold
- 본문 글자 크기: 최소 60pt (1080×1080 기준)
- 5단계 컬러 일관성 필수: 빨강·노랑·파랑·초록·보라
- 표지(1번)·결말(10번)은 강한 메시지, 중간(4~8번)은 단계별 학습
- 모든 카드 우측 하단에 작게 stopcall.kr 워터마크
