# 영업 키트 제작 가이드 (QR · 영상 · PDF)

공공기관 영업 메일에 첨부할 3종 세트를 만드는 법.

---

## 1. 제안서 PDF 만들기 ✅ (거의 완료)

### 준비된 것
- `assets/stopcall_proposal.html` — 인쇄 최적화 버전 (방금 생성)

### 변환 순서 (2분)
1. 방금 브라우저에 열린 `stopcall_proposal.html` 탭으로 이동
2. **⌘+P** (인쇄)
3. 대상: **PDF로 저장**
4. 설정:
   - 용지: A4
   - 여백: 기본
   - ✅ **배경 그래픽** 체크 (표·색상 유지)
   - 페이지: 전체
5. 저장 위치: `~/safe-smartact/assets/stopcall_proposal.pdf`
6. 저장

결과물 `stopcall_proposal.pdf`는 약 2페이지, 300~500KB 예상.

---

## 2. QR 코드 만들기 (3분)

### 가장 쉬운 경로 — QRCode Monkey
1. 👉 **https://www.qrcode-monkey.com**
2. 입력칸에 `https://stopcall.kr` 입력
3. 설정:
   - **Set Colors:** Foreground `#1f6feb` (브랜드 블루), Background 흰색
   - **Add Logo Image:** 스마택트 로고 PNG 업로드 (없으면 생략)
   - **Customize Design:** 네모 모서리 둥글게 (선택)
4. **Create QR Code** 클릭
5. **Download PNG** (최소 1000×1000px)
6. 저장: `~/safe-smartact/assets/qr.png`

### 네이버 QR 생성기 (대안)
👉 https://qr.naver.com — 더 단순, 로고 삽입은 불가

### 확인
QR을 핸드폰 카메라로 찍어서 `https://stopcall.kr`로 바로 열리는지 테스트.

### 활용
- 제안서 PDF 우측 상단 (이미 자리 잡혀있음 — QR 이미지만 교체)
- 활동북 표지·뒷면
- 명함 (추가 제작 시)
- 복지관 입구 스탠드형 포스터

## 3. 시연 영상 1분 녹화 (10분)

### 준비물
- Mac 기본 기능 (⌘+Shift+5)
- 브라우저 창 하나 (크롬·사파리)
- `stopcall.kr` 열어두기

### 녹화 순서
1. 브라우저를 **너무 크지도 작지도 않게** 조정 (폭 500~600px, 핸드폰 폭 흉내)
2. **⌘+Shift+5** → "선택한 부분 기록" → 브라우저 영역만 드래그
3. **옵션:**
   - 저장 위치: `~/safe-smartact/assets/`
   - 마이크: **꺼둠** (해설은 나중에 자막으로)
4. **기록** 클릭

### 시나리오 (약 50초)
```
0:00 - 0:05   홈 화면 "속지 않는 연습, 오늘 1분만 해볼까요?"
0:05 - 0:08   "▶ 연습 시작하기" 클릭
0:08 - 0:15   검찰사칭 전화 인트로 화면
0:15 - 0:18   "✅ 받기" 클릭
0:18 - 0:28   통화 중 + 4가지 선택지 천천히 보여주기
0:28 - 0:32   "③ 그냥 끊는다" 클릭
0:32 - 0:40   피드백 화면 (잘하셨어요! 메시지 확인)
0:40 - 0:45   "결과 보기" 클릭
0:45 - 0:55   결과 화면 + 핵심 원칙 카드 보여주기
0:55 - 1:00   stopcall.kr URL 강조하며 마무리
```

### 녹화 완료 후
1. 파일 확인: `assets/screen_recording_*.mov`
2. 이름 변경: `stopcall_demo_60s.mov`
3. (선택) iMovie·Quicktime으로 시작·끝 트림
4. (선택) 파일 크기 줄이기 — Quicktime → 파일 → 내보내기 → 720p

### 대안: 화면 캡처 GIF (용량 작음)
복잡하다면 GIF로 대체 — **Kap** (https://getkap.co/) 앱 설치 후 녹화 → GIF 저장 → 메일 본문에 바로 첨부.

## 4. 영업 키트 완성 후 폴더 구조

```
~/safe-smartact/assets/
├─ stopcall_proposal.html         ← 편집용 원본
├─ stopcall_proposal.pdf          ← 메일 첨부용 ⭐
├─ qr.png                         ← QR 이미지 ⭐
└─ stopcall_demo_60s.mov          ← 시연 영상 ⭐
```

이 3개 파일이 메일 첨부 표준 세트입니다.

## 5. 이메일 발송 전 최종 체크

- [ ] PDF 2페이지 이내인지 (넘치면 폰트 축소)
- [ ] QR 스캔 테스트 (핸드폰 카메라)
- [ ] 영상 재생 확인 (다른 사람 맥에서도 재생되는지)
- [ ] 수신자 이름·직책 정확한지
- [ ] 제안서 연락처: hello@stopcall.kr 로 통일됐는지
- [ ] 보내는 시간: 평일 오전 9~11시 또는 오후 2~4시 (열람률 높음)

## 💡 영상 대신 GIF 쓰는 팁

공공기관 담당자 메일함은 첨부파일 10MB 제한이 흔합니다.
MOV 파일이 크면 GIF로 변환하거나, **Vercel의 자동 생성 OG 이미지** 를 활용하세요:
- 메일 본문에 `https://stopcall.kr` 링크만 넣으면 미리보기 카드가 자동 생성됩니다
- 클릭 한 번이면 바로 체험 가능 (영상보다 오히려 전환율 높음)
