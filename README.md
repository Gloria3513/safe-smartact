# safe.smartact.kr — 멈춤연습 (MVP1)

디지털포용법 개정안(2026-03-12 국회 통과) 대응 시니어 사이버안전 교육 서비스. 스마택트 제작.

## MVP1 범위 (이 폴더)

- `index.html` — 단일 파일, 로그인·DB·빌드 없음
- 홈 → 전화 연습(검찰사칭 1시나리오) → 피드백 → 결과

## 띄우는 법

**로컬 확인**
```bash
open index.html
# 또는
python3 -m http.server 5173
```

**배포**
- Cloudflare Pages: 이 폴더를 그대로 드래그 업로드
- 또는 book.smartact.kr 하위 `/safe`로 우선 올려도 됨
- 최종 목표 도메인: `safe.smartact.kr`

## 다음 단계

- MVP2: 문자(스미싱) / 카톡(메신저피싱) / 링크(악성앱) 시나리오 각 3개
- MVP3: `/pro` 강사 모드 + KPI 리포트 (B2G 제안용)
- 확장: AI 음성복제 오디오 시나리오

## 디자인 원칙

- 본문 20px+, 버튼 72px+ (시니어 접근성)
- 실패 피드백에 비난 톤 금지
- 회원가입 없음 — QR 찍으면 바로 연습
